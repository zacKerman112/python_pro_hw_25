# Task 3 — Auth Chat

Чат з автентифікацією:

- у повідомленнях показується ім'я користувача;
- гості можуть лише читати чат;
- авторизовані користувачі можуть писати повідомлення.

## Запуск

```powershell
cd task_3
..\venv\Scripts\python.exe manage.py migrate
..\venv\Scripts\python.exe manage.py runserver
```

Потім відкрий `http://127.0.0.1:8000/`.

Для тесту:
1. відкрий чат як гість — поле вводу буде заблоковане;
2. зареєструйся або увійди;
3. відкрий другу вкладку і перевір, що повідомлення показуються з іменем користувача.
